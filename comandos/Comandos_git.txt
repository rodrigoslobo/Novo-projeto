
#####COMANDOS DO GIT#####

**Git= *Sistema de versionamento***

**GitHub = *Plataforma que faz uso desse sistema e armazena os códigos***

-----------------------------------
`git init` = *Inicializa um repositório GIT no projeto local, criando uma pasta oculta .git dentro do diretório do projeto*

`git branch -M main` = *Define uma Branch* 

`git add remote origin https://github.com/YOUR-USERNAME/YOUR-REPOSITORY` *Conecta a um repositório remoto*

`git branch` = *Exibe suas branches locais e informa qual está selecioanda no momento*

`git checkout <nome_da_branch>` = *Seleciona uma branch*

`git checkout -b <novo_nome_da_branch>` = *Cria uma nova branch e já muda para ela*

`git add .\README.md` = *Envia arquivo específico ao repo remoto*

`git add .` = *Envia todos os arquivos ao repo remoto*

`git status` = *Mostra se há alguma atualização não sincronizada com o remoto*

`git commit -m "Adiciona projeto inicial"` = *Faz um commit*   

`git remote` = *Exibe o nome do repo remoto*

`git pull origin main` = *Baixa as alterações do remoto para o local*

`git push origin <sua_branch>` = *Envia as alterações do local para o remoto*

`git -D <sua_branch>` = *Deleta a branch local*

`git push origin <sua_branch> --delete` = *Deleta a branch no github*

`git log` = *Exibe log de commits realizados*

`git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY` = *Clona determinado repositório para a sua máquina*

`git merge <sua_branch>` = Mescla banches

`git tag -a v1.0 -m "Versão 1.0"` = *Cria uma nova TAG*

`git push origin v1.0` = *Envia determinada tag para o remoto*

`git tag` = *Exibe as tags*
